A Geographical area with data centers is called an AWS Region. Each region has more than a single availability zone. A single data center or a combination of many data centers within an AWS region is called an Availability Zone. Each AZ is connected to a separate network and power for high availability.

Splitting a single dataset across many servers is called Sharding.
